Project Objective

Create a firework simulation to demonstrate a well-rounded understanding of the C as a language.

How To Use

Unzip the project zip into a folder in a Linux environment. Once unzipped there should be the Lab4 executable and several example firework files (.fwx) to serve as input. Then all you need to do as shown in the video above is execute the firework simulation with a given file with a command like “lab4 stars.fwx.” This will open a visual of the fireworks being shot off.

Key Features

Reading of input files
If you want to make your own firework shows you can always create your own fwx files. FWX files are text files with each line representing all the data necessary to launch a firework in the simulation. An example valid fwx line looks like the following line:

 0.0  F70A87BA  7.209  -140.0  39.7  113.44

The data in order from left to right are the time the firework is launched, the firework code (explained in detail later), the fuse duration, the initial x position where the firework is launched from (0.0 being the center of the screen), the x velocity and the y velocity.

The firework code is made up of hexadecimal values that represent qualities not related to the firework’s motion. The code is structured like this:

 E C DD M YYY

E is 3 Boolean bits that represent from left to right whether the fireworks have a star effect, whether it has a flash effect, and whether it has a boom sound effect. The last bit of E is not used currently and remains for cleanness of the firework code’s structure. The next 4 bits are the Color of the firework. And the rest of the firework data is the day, month, and year the firework was made. This is used to simulate data necessary if the firework experiences an issue such as hitting the ground without going off. This data will output in the little console at the bottom of the visual output.

Putting in values for any of these numbers could result in a firework that does not explode properly such as seen in the example “unsafe” FWX file which shows two fireworks exploding at launch and one hitting the ground, so build fireworks carefully!
Memory management
An implementation of a Linked lists that is used to read the fireworks. We can then use the list nodes to launch them in time according to the launch time present for that firework in the fwx file. The implementation is complete with malloc and releasing of list nodes which consequently frees the information related to fireworks.
Physics
All the velocity and position numbers are plugged into the basic equations of motion to find the next position of the firework using real-time. These calculations are done on each update tick (delta T) determined by provided animation library along with the y velocity (VT) of fireworks in the input. Equations used for physics are:
  o   X’ = X + VX *dt
  o   Y’ = Y + VY * dt + 0.5 * GRAVITY * dt * dt
  o   VY = VY + GRAVITY * dt
GRAVITY = -32.17405 feet per second per second
